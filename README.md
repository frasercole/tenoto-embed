# TENOTO embed

Animated TENOTO wave logo, hosted as a static page for embedding in Notion (or anywhere that accepts an iframe URL).

- **`index.html`** — the slim, self-contained embed widget. Centers itself in whatever container it's given.
- **`playground.html`** — the full editor with controls for tuning the animation.

## Live

- Embed: https://frasercole.github.io/tenoto-embed/
- Playground: https://frasercole.github.io/tenoto-embed/playground.html

## Embedding in Notion

In your Notion page, type `/embed`, paste `https://frasercole.github.io/tenoto-embed/`, and resize the block to taste.
